# Raising the Stone

No iron. No pulleys. No electricity. An interactive physics simulator that walks through the mathematics of building the Great Pyramid — one block at a time, all 201 courses.

**Features:**
- Timeline mode: watch all 201 courses rise with live force calculations
- Single Block mode: test any mass, ramp angle, and surface condition
- The Scribe's Reckoning: detailed math ledger for every course
